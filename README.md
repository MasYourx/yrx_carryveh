# yrx_carryveh

A Simple Script That Admin Can Be Able To Carry All Vehicles

## Supported Frameworks

- ESX

## Dependencies

- [ox_lib](https://github.com/overextended/ox_lib)

## Command Usages

- /carryvehicle --To carry nearest vehicle
- /dropvehicle -- To drop carried vehicle
